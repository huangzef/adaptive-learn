## 项目信息

- **项目类型**：编程/调研/设计/理论
- **项目名称**：[名称]
- **项目目标**：[目标描述]
- **难度评级**：入门/基础/进阶/挑战
- **子任务**：见下方列表
  - [ ] 子任务1
  - [ ] 子任务2
- **项目进度**：X% / [已完成子任务数/子任务总数]
- **交付物**：[产出物列表]
- **当前阶段**：规划/执行/收尾

### 说明

| 字段 | 说明 |
|:-----|:-----|
| 项目类型 | 项目的性质类型：编程（代码开发）、调研（资料收集分析）、设计（UI/架构设计）、理论（证明推导） |
| 项目名称 | 项目的名称或标题 |
| 项目目标 | 项目要达成的核心目标或产出 |
| 难度评级 | 综合考虑复杂度、所需知识深度、时间投入的难度评估 |
| 子任务 | 将项目拆解为的具体可执行任务，使用 checkbox 标记完成状态 |
| 项目进度 | 通过已完成子任务数/子任务总数计算的百分比，反映项目整体完成度 |
| 交付物 | 项目完成时需要交付的具体产出物列表，如代码文件、文档、报告、演示链接等 |
| 当前阶段 | 项目所处的主要阶段：规划（方案设计）、执行（具体实施）、收尾（整理完成） |
| 项目反馈 | 项目完成后的反馈记录，用于更新学习者画像（见下方反馈格式） |

### 项目反馈格式

项目完成后，将以下内容追加到 `../../learning_profiles/{user_id}.md`：

```markdown
## 项目记录
- 日期：{日期}
- 项目类型：{项目类型}
- 项目名称：{项目名称}
- 难度：{难度评级}
- 自我评分：{1-5}
- 用时：{小时}
- 产出：{交付物}
```
