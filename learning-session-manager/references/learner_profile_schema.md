> 所有参数公式详见 `continuous_scoring_engine.md`，此处仅列出参数名称和用途。

| `颗粒度` | 15-120 min | 建议的单个任务时长 |

## 更新规则

### 每次教学后自动更新
1. **知识点掌握度**：基于里程碑完成情况更新
2. **学习风格**：从互动模式中推断并记录
3. **常见困惑**：检测用户反复卡住的概念

### 画像加载时机
- 新会话开始时
- 用户说"继续学习"时
- 切换教学主题时

### 画像应用方式
- 加载后注入为上下文
- 指导教学深度选择
- 调整问题类型比例
- 个性化鼓励语言

## 成长指标

> 由 learning-session-manager 在每次会话结束时自动更新。
> Project Tracker 在项目完成时同步写入项目用时。

| 字段 | 类型 | 说明 |
|:-----|:-----|:-----|
| `estimated_speed` | 知识点/小时 | 基于最近 N 次会话估算的平均学习速度 |
| `consistency_score` | 1–10 | 学习频次稳定性评分（参考：每周学习天数占比） |
| `strongest_learning_pattern` | 视觉型 / 听觉型 / 动觉型 / 理论型 | 从会话行为模式中推断的主导学习风格 |
| `last_calculated` | YYYY-MM-DD | 上次指标更新日期 |

### 计算规则

- **estimated_speed**：会话结束时，用本次掌握的知识点数量 ÷ 本次学习时长（小时），滚动平均至最近 3 次有效会话
- **consistency_score**：基于 `## 学习历史摘要` 中最近 8 周的周活跃天数计算，公式：活跃周数 / 8 * 10，取整
- **strongest_learning_pattern**：从本次会话的行为信号推断（偏好看图表→视觉型，偏好讨论→听觉型，偏好动手实验→动觉型，偏好推导证明→理论型），出现频次最高的类型记录
